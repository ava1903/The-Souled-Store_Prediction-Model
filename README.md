# 🎭 The Souled Store — Data-Driven Growth Dashboard

An interactive Streamlit dashboard applying **Barış Karaman's 9-article Data-Driven Growth with Python** framework to The Souled Store, India's leading D2C pop culture merchandise brand.

## 📊 Dashboard Sections

| Section | Articles Used | What It Shows |
|---|---|---|
| 🏠 Company Overview | Article 1 | North Star Metric, revenue trends, customer base |
| 🎯 Solution 1: Cross-Fandom Discovery | Articles 2, 3, 5 | RFM segments, Fandom Breadth Score, LTV comparison, fandom transitions |
| 📉 Solution 2: Churn Prevention | Articles 2, 3, 4, 5, 8 | Churn risk distribution, buyer personas, Uplift scores, LTV-weighted budget |
| 📈 Solution 3: Revenue Diversification | Articles 6, 7, 8, 9 | LSTM baseline vs actual, offer ROI comparison, A/B test tracker, membership funnel |

> **Note:** All data is simulated for demonstration purposes.

---

## 🚀 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/tss-dashboard.git
cd tss-dashboard

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```

---

## ☁️ Deploy on Streamlit Cloud (Free)

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click **New app**
4. Select your repo → branch `main` → file `app.py`
5. Click **Deploy** — done ✅
